####事件简介

&emsp;好近官网是用wordpress简单修改的,本来是用的域名是`www.haojin.in`,后来打算启用`haojin.in`,更酷,哈哈.本来以为修改了nginx的`rewrite`配置后就ok了,擦,谁知道不行....

&emsp;修改后的部分nginx配置:

```
server {
                listen      80;
                server_name     d.haojin.in;
                rewrite ^(.*) http://haojin.in/?from=d permanent;
            }

server {
                listen      80;
                server_name     www.haojin.in;
                rewrite ^(.*) http://haojin.in permanent;
            }

server {
            listen     80;
            server_name  haojin.in;
	}
```

发现不行耶,后来钟大啪啪啪修改了几下wordpress的mysql配置就OK了,

```
mysql> select * from wp_options where option_value like '%fengmi%';
+-----------+-----------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+----------+
| option_id | option_name     | option_value                                                                                                                                                                                                                                                                                                                                                                                                        | autoload |
+-----------+-----------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+----------+
|         1 | siteurl         | http://www.haojin.in                                                                                                                                                                                                                                                                                                                                                                                            | yes      |
|         2 | home            | http://www.haojin.in

```

修改了表`wp_options`中字段`option_name`中`siteurl`和`home`的值,变成`http://haojin.in`就好了.蛋疼~
