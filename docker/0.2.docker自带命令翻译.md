1. docker ps命令

```
docker ps --help

-a 显示所有的容器,默认docker ps命令只显示正在运行的容器.
-q 只显示容器的ID.
-s 显示所占空间大小。

```

2. `docker run`

```
docker run [OPTIONS] image [COMMAND] [ARG]

-a,--attach=[] attach to STDIN,STDOUT or STDERR,默认包括所有。可以单独指定,一般不使用。
docker run -a stdin -a stdout -it centos:6 /bin/bash 
--add-host 添加主机→IP的映射到容器的hosts文件中。
docker run --add-host google:8.8.8.8 centos:6 /bin/bash
```
