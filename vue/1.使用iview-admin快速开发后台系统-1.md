#### 安装环境

`iview admin`需要有nodejs环境, 由于nodejs版本多样, 我采用的是nvm的方式安装nodejs, 具体安装nvm的方法请参考官方文档, 咱们能用到的只有下面2个命令, 安装的是10.X版本的nodejs:

```bash
#nvm install 10
#nvm use 10
```

最好能把第二个命令写到`.zshrc`文件中, 这样不用每次手工执行了.
