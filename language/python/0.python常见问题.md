1. 修改了supervisor的配置后重启了supervisord进程,旧程序依旧存在,所以旧程序依旧占用一些端口等资源.举例,一次`near-api`线下测试环境的部署.

3. 使用`pep8`检查代码风格

```
pip install pep8

import os, sys
         ^
    Imports should usually be on separate lines.

    Okay: import os\nimport sys
    E401: import sys, os
```
